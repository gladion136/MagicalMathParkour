# Magical Math Parkour

Lernspiel zum Üben von Linearen, Quadratischen und Sinus Funktionen.

Entstanden während des Modules "MINT- In Praxis und Lehre" an der Technischen Hochschule Ostwestfahlen-Lippe.

Autoren:
- Etienne Schmidt
- Jan Lüking
- David Schelinski
